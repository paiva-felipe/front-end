> Repositório central de estudos e projetos práticos desenvolvidos na disciplina de **Programação Front-End**.

Este espaço é dedicado à construção de interfaces modernas, focando na transição do design estático para aplicações interativas, dinâmicas e totalmente responsivas.

---

## 🎯 Objetivos do Repositório

- [x] Dominar a estrutura semântica com **HTML5**.
- [x] Criar layouts flexíveis e responsivos com **CSS Grid e Flexbox**.
- [x] Implementar lógica e interatividade real através da **Manipulação do DOM**.
- [x] Aplicar boas práticas de organização de código e **Autogestão**.

---

## 📂 Estrutura das Aulas

O aprendizado é dividido em módulos evolutivos:

### 📗 Módulo: Design e Fluidez
* **Aula 04:** Design Responsivo, Viewport e Media Queries (O conceito de adaptação inteligente).

### 📘 Módulo: Dinamismo com JavaScript
* **Aula 05: DOM Estático:** Seleção de elementos com `querySelector`, alteração de conteúdos e manipulação de classes.
* **Aula 06: DOM Eventos:** Transformando a página em uma aplicação reativa com `addEventListener`, eventos de mouse e cálculos em tempo real.

## 🚀 Projetos em Destaque

### 🍔 TechFood (Live Code)
*Projeto modelo para demonstração de conceitos.*
- **Funcionalidade:** Simulador de pedidos com cálculo dinâmico de subtotal, formatação de moeda e interface adaptável para dispositivos móveis.

### 🌿 EcoCycle (Desafio Prático)
*Projeto para aplicação de conhecimentos pelos alunos.*
- **Funcionalidade:** Portal de notícias ecológicas com simulador de economia de recursos naturais e player de vídeo interativo via JS.
