Geek Store

Esta atividade foi proposta para terstarmos nossas habilidades referentes aos ultimos conteúdos aprendidos, para isso tivemos que formular um arquivo js com manipulação e eventos DOM. 
 Esse arquivo trouxe interatividade ao sistema que  antes só possuia interface gráfica, traxendo mais conforto ao uso do site Geek Store, que inclusive é otimo!

🛠️ Tecnologias Utilizadas
Usamos Linguagens de marcação HTML para criar o layout da pagina (textos, botões, sessões...) e CSS para estiliza-los.
Para a interatividade usamos Java Script inegrado com os componentes para dar ações a eles.
Alem disso uma estrura organizada dentro do projeto, separando os scripts, styles e imagens dentro do source.
